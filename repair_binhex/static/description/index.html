Odoo 12 Reparation module

Allows you to manage the repair of external products with the repair module


Note: The translations of this module are currently in process and will be activated in its next version.

Features

  • Repair external products.
  • Creates an entry and exit stock piking for the product to be repaired
  • Can sign and add note to the prints of stock pinking operations.
  • Can invoice the parts you need for repair and services such as the workmanship.

Usage

Fisrt go to reparations menu.

Create in reparation product or select an exitence product.

Note: As you can see when installing it, it creates a location and repair stock operations for input and output.

If you create the product need to be storable or make and error.

Here you can add the repair parts to invoice or budget them.

You can add signature to the prints of stock pinking operations.

You can add notes to the prints of stock pinking operations.

When you confirm the repair, the stock picking will be created and you will be able to access it through an icon at the top if you access it you will be able to print it.

When you print it you can deliver it to the client with his signature and the notes.
